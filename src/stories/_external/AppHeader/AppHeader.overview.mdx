  <h2 id="when-not-to-use" style={{ position: 'relative' }}>‚ùå When not to use</h2>
  <div>AppHeader is an application-level navigation shell, not a general-purpose header.</div>
  <ul> 
    <li>Do not use AppHeader for simple in-page headers (e.g., section titles, page banners). Use Banner or a semantic h1 instead.</li> 
    <li>Avoid it in micro-frontends or isolated widgets where global navigation isn‚Äôt required.</li> 
    <li>Don‚Äôt use it if you only need a standalone navigation menu; use HeaderNavigation or SideNav directly from Carbon.</li> 
    <li>If you need a marketing/landing page hero (visual focus, imagery, CTAs), use a Banner or custom hero component instead.</li> 
  </ul>
