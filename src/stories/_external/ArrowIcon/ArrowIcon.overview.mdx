  <h2 id="purpose-what-is-it" style={{ position: 'relative' }}>Purpose / What it is?</h2>  
  <div>The ArrowIcon component provides a reusable wrapper for displaying directional arrow icons from the Carbon icon set. It standardizes icon size, accessibility attributes, and className handling, ensuring consistent use of arrow symbols across the application.</div>

  <Primary /> 
</div>


<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="when-to-use" style={{ position: 'relative' }}>✅ When to Use</h2> 
  <ul>
    <li>To indicate navigation or direction (e.g., “Read more”, “Next step”).</li> 
    <li>When linking to another page, section, or external resource.</li> 
    <li>To visually reinforce call-to-action buttons or links.</li>  
  </ul>
</div> 



<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="when-not-to-use" style={{ position: 'relative' }}>❌ When not to use</h2>
  <ul>
    <li>As a decorative element without meaning (icons should support, not replace, text).</li> 
    <li>For complex navigation patterns — use Carbon’s Link or Button components instead.</li> 
    <li>When another icon communicates the intent more clearly (e.g., Download, Upload, ChevronDown).</li>  
  </ul>
</div>

  </li>

  <li data-start="1037" data-end="1092">
