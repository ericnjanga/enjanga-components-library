  <h2 id="purpose-what-is-it" style={{ position: 'relative' }}>üìå Purpose / What it is?</h2>

  <h2 id="when-to-use" style={{ position: 'relative' }}>‚úÖ When to Use</h2> 
  <ul data-start="651" data-end="921">
  <li data-start="651" data-end="753">
  <p data-start="653" data-end="753">When you need to display a pictogram (illustration/icon) to support or enhance meaning in content.</p>

  <li data-start="754" data-end="842">

  </li>
  <li data-start="843" data-end="921">
  <p data-start="845" data-end="921">To improve scannability of content-heavy sections with supporting visuals.</p>
  </li>
  </ul>
</div>



<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="when-not-to-use" style={{ position: 'relative' }}>‚ùå When not to use</h2>
  <ul data-start="955" data-end="1310">
  <li data-start="955" data-end="1099">
  <p data-start="957" data-end="1099">Do not use <code data-start="968" data-end="985">CustomPictogram</code> as a replacement for functional icons (like buttons, actions, or navigation). Use the <code data-start="1072" data-end="1078">Icon</code> component instead.</p>
  </li>
  <li data-start="1100" data-end="1224">
  <p data-start="1102" data-end="1224">Avoid using pictograms as the <strong data-start="1132" data-end="1144">only way</strong> to communicate meaning ‚Äî always pair with text for accessibility and clarity.</p>
  </li>
  <li data-start="1225" data-end="1310">
  <p data-start="1227" data-end="1310">Do not use pictograms as background decoration unless they serve a clear purpose.</p>
  </li>
  </ul>
</div>


<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="features" style={{ position: 'relative' }}>‚ú® Features</h2>
  <ul data-start="1337" data-end="1728">
  <li data-start="1337" data-end="1404">
  <p data-start="1339" data-end="1404">Dynamically loads pictograms by name through a helper function.</p>

  <li data-start="1494" data-end="1547">
