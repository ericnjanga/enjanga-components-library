  <h2 id="property-validation" style={{ position: 'relative' }}>Property validation</h2> 

  <div>
    <div>
    <strong data-start="434" data-end="472">Rule 1: Tile mode (Link vs. Modal)</strong>
    </div>
    
    <div>
    A <code data-start="477" data-end="489">CustomTile</code> must be either:
    </div>
  </div>


  <ul data-start="508" data-end="636" style={{ marginTop: '-0.8rem', marginBottom: '-0.8rem' }}>
  <li data-start="508" data-end="578">
  <p data-start="510" data-end="578"><strong data-start="510" data-end="525">A link tile</strong> ‚Üí Provide <code data-start="536" data-end="545">linksTo</code> (and optionally <code data-start="562" data-end="574">linkTarget</code>).</p>
  </li>
  <li data-start="579" data-end="636">
  <p data-start="581" data-end="636"><strong data-start="581" data-end="597">A modal tile</strong> ‚Üí Provide <code data-start="608" data-end="633">modalIsAvailable={true}</code>.</p>
  </li>
  </ul>


  <div style={{ marginBottom: '30px' }}>
   üö´ <strong data-start="641" data-end="672">Never both at the same time</strong> ‚Äî a <code data-start="677" data-end="689">CustomTile</code> cannot act as a link and modal trigger simultaneously. 
  </div>
 



  <div>
    <div>
      <strong data-start="753" data-end="783">Rule 2: Modal content type</strong>
      </div>
  When <code data-start="791" data-end="816">modalIsAvailable={true}</code>, you must provide <strong data-start="835" data-end="850">exactly one</strong> type of modal description: 
  </div>


  <ul data-start="880" data-end="959" style={{ marginTop: '-0.8rem', marginBottom: '-0.8rem' }}>
  <li data-start="880" data-end="918">
  <p data-start="882" data-end="918"><code data-start="882" data-end="905">modalPlainDescription</code> <em data-start="906" data-end="916">(string)</em></p>
  </li>
  <li data-start="919" data-end="959">
  <p data-start="921" data-end="959"><code data-start="921" data-end="943">modalRichDescription</code> <em data-start="944" data-end="957">(rich text)</em></p>
  </li>
  </ul>

  <p data-start="961" data-end="1042">üö´ <strong data-start="964" data-end="986">Never provide both</strong> ‚Äî only one description prop should be used at a time.</p>


</div>
