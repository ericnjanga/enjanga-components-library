  <h2 id="property-validation" style={{ position: 'relative' }}>Property validation</h2> 
  <h3>Property Validation 1</h3>
  <div>CustomTile must be either (But cannot be both at the same time):</div>
  <ul>
    <li>A link (provide 'linksTo' and optionally 'linkTarget')</li>
    <li>A modal trigger (provide 'showsModal')</li>
  </ul>

  <h3>Property Validation 2</h3>
  <div>Only one property for the modal's main content can be provided at the time:</div>
  <ul>
    <li>Either plainDescription (string)</li>
    <li>Or richDescription (rich text)</li>
  </ul>
</div>
