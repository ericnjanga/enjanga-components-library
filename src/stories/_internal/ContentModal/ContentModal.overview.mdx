  <h2 id="purpose-what-is-it" style={{ position: 'relative' }}>Purpose / What it is?</h2>  
  <div>The <code>ContentModal</code> component is a wrapper around Carbon’s <code>ComposedModal</code>. It provides a standardized, accessible modal dialog with a header, body, and footer. It allows you to display additional content, forms, or actions without leaving the current page.</div>

  <Primary /> 
</div>
  

<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="when-to-use" style={{ position: 'relative' }}>✅ When to Use</h2> 
  <ul data-start="451" data-end="741">
  <li data-start="451" data-end="521">
  <p data-start="453" data-end="521">To present critical information that requires user acknowledgment.</p>
  </li>
  <li data-start="522" data-end="631">
  <p data-start="524" data-end="631">For workflows that need focused user attention, such as confirming an action or filling out a short form.</p>
  </li>
  <li data-start="632" data-end="741">
  <p data-start="634" data-end="741">To display secondary content without navigating away (e.g., terms and conditions, detailed descriptions).</p>
  </li>
  </ul>
</div>



<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="when-not-to-use" style={{ position: 'relative' }}>❌ When not to use</h2>
  <ul data-start="770" data-end="1072">
  <li data-start="770" data-end="882">
  <p data-start="772" data-end="882">For lightweight interactions like tooltips, inline popovers, or dropdowns (use smaller UI patterns instead).</p>
  </li>
  <li data-start="883" data-end="1000">
  <p data-start="885" data-end="1000">To display large or complex content that requires scrolling through multiple screens (consider a dedicated page).</p>
  </li>
  <li data-start="1001" data-end="1072">
  <p data-start="1003" data-end="1072">As the primary way to navigate between pages or core user journeys.</p>
  </li>
  </ul>
</div>


<div style={{ position: 'relative', paddingTop: '50px', marginTop: '-50px', zIndex: '1', marginBottom: '60px' }}>
  <h2 id="features" style={{ position: 'relative' }}>✨ Features</h2>
  <ul data-start="1094" data-end="1544">
  <li data-start="1094" data-end="1166">
  <p data-start="1096" data-end="1166">Based on Carbon’s <code data-start="1114" data-end="1129">ComposedModal</code> for accessibility and consistency.</p>
  </li>
  <li data-start="1167" data-end="1236">
  <p data-start="1169" data-end="1236">Configurable header with label, heading, and optional subheading.</p>
  </li>
  <li data-start="1237" data-end="1319">
  <p data-start="1239" data-end="1319">Flexible body content (<code data-start="1262" data-end="1272">children</code>) for rich text, forms, or custom components.</p>
  </li>
  <li data-start="1320" data-end="1375">
  <p data-start="1322" data-end="1375">Footer with a customizable secondary action button.</p>
  </li>
  <li data-start="1376" data-end="1462">
  <p data-start="1378" data-end="1462">Controlled <code data-start="1389" data-end="1397">isOpen</code> prop and <code data-start="1407" data-end="1418">setIsOpen</code> callback for open/close state management.</p>
  </li>
  <li data-start="1463" data-end="1544">
  <p data-start="1465" data-end="1544">Medium (<code data-start="1473" data-end="1477">md</code>) size by default, with other sizes (<code data-start="1514" data-end="1518">xs</code>, <code data-start="1520" data-end="1524">sm</code>, <code data-start="1526" data-end="1530">lg</code>) available.</p>
  </li>
  </ul>
</div>
